TOOL // FUTURES SPEC SHEET

Futures Spec Sheet

The multiplier and tick value behind every point \u2014 the numbers you must have before sizing a futures trade.

Your stop distance in ticks \u00d7 the tick value = your dollar risk. Micro contracts (MES, MNQ, MCL\u2026) are 1/10 the size and exist so you can learn without full-size risk. Specs are standard references \u2014 confirm with your broker.

TOOL // EQUITY SIMULATOR

Equity Simulator

Five random runs of the same edge \u2014 same odds, different luck. Watch variance before it\u2019s your money.

Each line risks the same % per trade with the same odds \u2014 the spread between best and worst run is pure variance. If 1% risk already looks scary, imagine 10%.
